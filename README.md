# 💈 Barbearia do Roseira - Sistema de Seleção

Desafio prático de Front-end desenvolvido para a aula do Prof. Leonardo Rocha.

## 🚀 O que foi desenvolvido:
- Interface com cards de serviços (Corte, Barba e Sobrancelha).
- Lógica em JavaScript para somar os valores em tempo real.
- Layout responsivo com CSS.

## 🛠️ Requisitos Técnicos Aplicados:
- Manipulação de DOM (`querySelector`).
- Eventos de clique (`addEventListener`).
- Uso de `forEach` e `Arrow Functions`.
- Conversão de dados com `parseFloat`.
